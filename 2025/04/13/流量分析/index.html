流量分析

2025年04月13日

流量分析

源ip筛选：

  1. ip.src == 源ip地址
  2. 在Internet Protocol Version 4下的Source字段

筛选源IP的所有包：点击source字段，再选择过滤器应用，点击选中

‍

目的ip筛选：

  1. ip.dst ==目的ip地址
  2. 在Internet Protocol Version 4下的Destination字段，再选择作为过滤器应用，点击选中

‍

追踪流

右键追踪流TCP，看见png

保存原数据，用010对数据进行编辑，去掉png之前的内容，再进行保存

‍

数据提取：

自动提取通过http传输的内容

文件->导出对象->HTTP

打开后点击save将其保存到文件夹里

或者手动提取http传输的内容

在分组传输中找到data,Line-based text, JPEG File Interchange Format, data:text/html层。选中后导出分组字节流。在弹出的窗口中设置开始和结束的字节（原字节数开头加3，结尾减3）。

筛选/搜索
